# RegexRiddle - Progetto Esame Tecnologie Web di Cervera Francesco - CdL Informatica - Università Federico II 

## Cos'è ? 

RegexRiddle è un gioco che date delle stringhe di caratteri (anche alfanumerici) l'utente loggato deve indovinare l'espressione regolare associata che 
generi SOLO ED ESCLUSIVAMENTE i risultati ritenuti VALIDI; qualora la RegEx inserita dall'utente dovesse generare anche stringhe NON VALIDE, la giocata
sarà considerata PERDENTE
